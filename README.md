# 🚆 RailDocs AI

RailDocs AI is an AI-powered document management system designed for railway and metro operations. It helps staff upload, manage, and summarize operational documents efficiently.

---

## 🚀 Features

* 🔐 JWT-based Authentication
* 📄 Document Upload & Management
* 🤖 AI Summary using Gemini API
* 🏢 Department-based Access Control
* 📊 Dashboard with document stats
* 📅 Meeting Management
* 🔔 Notifications

---

## 🧠 Tech Stack

* Frontend: React Native (Expo)
* Backend: Node.js, Express.js
* Database: MongoDB
* AI: Google Gemini API

---

## ⚙️ Setup

### Backend

```bash
npm install
npm run dev
```

### Frontend

```bash
npm install
npx expo start
```

---

## 🎯 Description

A smart system that reduces manual work by digitizing documents and generating AI-based summaries while ensuring secure department-level access.
